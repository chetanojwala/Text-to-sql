# 🤖 AI-Powered Text-to-SQL System

## 📌 Overview
This project converts natural language queries into SQL queries and retrieves results from a database.

It allows users to interact with databases without writing SQL manually.

---

## 🚀 Features
- Natural language → SQL conversion  
- AI-powered query generation  
- SQLite database integration  
- Dynamic query execution  

---

## 🧠 How It Works
1. User enters a query  
2. AI converts it into SQL  
3. SQL runs on database  
4. Results are displayed  

---

## 💡 Example Queries
- Show all students  
- Students with marks above 80  
- Who is the top student  
- How many students are there
